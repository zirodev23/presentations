Javascript Pamati

1. uzdevums: DOM elementi un notikumu klausītāji

  1. Izveido jaunu HTML projektu "html-dom-js" ar 'index.html' failu
  2. Izveido text tipa input lauku un pogu
  3. index.html failā izveido sadaļu script Javascript koda sadaļai.
  4. Izvadi savu vārdu Javascript konsolē pārliecinoties, ka kods tiek izpildīts.
  5. Ar Javascript palīdzību atrast input elementu un izvadīt tā vērtību konsolē.
  6. Izveidot funkciju printName, ka izvada tavu vārdu konsolē.
  7. Izveidot notikumu klausītāju (event listener), kas klausās un spēj reaģēt un pogas klikšķi. Izsaukt printName funkciju, kad uz pogas tiek veikts klikšķis (click).
  8. Uzlabot notikumu klausītāju, lai printName funkcijas vietā tiktu izsaukta cita funkcija, kas izvada input lauka vērtību.

2. uzdevums: Lietotāja ievades apstrāde

  1. Izveido jaunu failu loop.html, kurā ir divi skaitļu ievades lauki un poga.
  2. Pirmā ievades lauka vērtībai ir jābūt mazākai par otrā ievades lauka vērtību. Abiem skaitļiem ir jābūt naturāliem.
  3. Izveidot pogas klikšķa notikuma klausītāju, kas pārbauda vai skaitļi ir pareizi ievadīti. Pretējā gadījumā izdot alert paziņojumu.
  4. Ja skaitļi ir ievadīti pareizi, tad izvadīt konsolē visus naturālos skaitļus, kuri atrodas starp šiem diviem skaitļiem.
  5. Izveidot paragrāfa elementu zem pogas un tajā izvadīt nepieciešamos skaitļus
