JavaScript Pamati

1. uzdevums: Vides sagatavošana

  1. Izveido mapi js-scripts savā varda mapītē
  2. Izveidoto mapi atver ar VS Code
  3. Izveido tajā failu output.js
  4. Izveidotajā failā ieraksti koda rindu, kas konsolē izvada tavu vārdu
  5. Terminal aplikācijā js-scripts mapē izpildi komandu node output.js

Turpmākajos uzdevumos izmantot node (nevis interneta pārlūku), lai izpildītu Javascript failus

2. uzdevums: Mainīgie

  1. Izveidot failu variables.js
  2. Izveidot string tipa mainīgo name, kurā tiek glabāts tavs vārds
  3. Izveidot number tipa mainīgo age, kurā tiek glabāts tavs vecums
  4. Izveidot boolean tipa mainīgo isStudent, kura vērtība ir true jeb patiesa
  5. Izveidot number tipa konstanti PI, kura vērtība ir 3.14

3. uzdevums: Masīvi un Cikli

  1. Izveidot failu arrays-loops.js
  2. Izveido divus for ciklus (loop), kuri izvada skaitļus no 11 līdz 20 un no 20 līdz 11 attiecīgi
  3. Izveidot masīvu jeb array tipa mainīgo, kurā tiek glabāti 5 tavu draugu vārdi
  4. Izvadi konsolē pirmā un trešā drauga vārdu
  5. Izveido mainīgo, kurā glabājas izveidotā masīva garums jeb elementu skaits tajā
  6. Izmantojot šo mainīgo, izveido ciklu, kas izvada katru vārdu jaunā rindā
  7. Papildini izveidoto ciklu ar numerāciju, piem., "1. Jānis", "2. Antra" utt.
