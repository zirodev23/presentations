Grade Calculator

Šajā uzdevumā jūs izveidosiet vienkāršu aplikāciju, kas aprēķina studenta vērtējumu, pamatojoties uz iegūtajiem punktiem un maksimālo punktu skaitu.

Instrukcijas

  1. Izveido HTML failu ar nosaukumu grade_calculator.html.
  2. HTML failā pievieno:
    • Virsrakstu: Grade Calculator.
    • Ievades laukus:
      • Vārds: tekstu ievades lauks.
      • Iegūtie punkti: skaitļu ievades lauks.
      • Maksimālais punktu skaits: skaitļu ievades lauks.
    • Izkrītošo izvēlni, lai izvēlētos Vērtēšanas shēmu:
      • Procentuālā vērtība (0-100).
      • Burtu vērtējums (A, B, C utt.).
    • Pogu Aprēķināt vērtējumu.
    • Sekciju, kurā tiks parādīts aprēķinātais vērtējums.

CSS un funkcionalitāte

1. Pievieno ārējo CSS failu, kas stilizē formu:

  • Forma jācentrē ekrānā un jānosaka konkrēts platums.
  • Ievades lauki aizņem visu formas platumu, bet nepārsniedz tās robežas.

2. Ar JavaScript pievieno šādu funkcionalitāti:

  • Izmanto addEventListener, lai aprēķinātu vērtējumu, kad tiek nospiesta poga.
  • Ja izvēlēta procentuālā vērtēšana:
    • Izrēķini: (Iegūtie punkti / Maksimālais punktu skaits) * 100.
  • Ja izvēlēts burtu vērtējums:
    • 90-100: A
    • 80-89: B
    • 70-79: C
    • 60-69: D
    • Zem 60: F
  • Parādi rezultātu lietotājam.

Validācija

Nodrošini, ka:

  • Iegūtie punkti nepārsniedz maksimālos punktus.
  • Ievades lauki nav tukši un satur tikai pareizus datus.
