Ievads HTML

HTML jeb Hiperteksta iezīmēšanas valoda ir tīmekļa vietņu pamats. Tas nosaka tīmekļa lapas struktūru un saturu.

Elementi un tagi

HTML sastāv no tagiem. Populārākie tagi:

  • <h1>-<h6>: virsraksti
  • <p>: paragrāfs
  • <a>: hipersaite
  • <img>: attēls

HTML piemērs

Vienkāršs HTML kods:

<h1>Sveiki, pasaule!</h1>
<p>Šis ir vienkāršs paragrāfs.</p>
<a href="https://example.com">Apmeklē mūsu lapu</a>

Uzdevums: Iepirkumu saraksts un preču apraksti

Mērķis: izveidot HTML dokumentu, kas parāda izpratni par numurētiem (ordered), nenumurētiem (unordered) un apraksta sarakstiem (description list).

Norādījumi

  • Izveido pamata HTML lapas struktūru ar <title> "Mans iepirkumu saraksts."
  • HTML dokumenta <body> daļā izveido virsrakstu <h1> ar nosaukumu "Iepirkumu saraksts."
  • Zem virsraksta izveido nenumurētu sarakstu (unordered list) ar vismaz 5 lietām, ko nepieciešams iegādāties veikalā.
  • Vienam no saraksta elementiem pievieno ligzdotu (nested) sarakstu – numurētu sarakstu (ordered list), kas norāda darbības, kā izmantot vai sagatavot konkrēto preci.
  • Pievieno vēl vienu virsrakstu <h2> ar nosaukumu "Prioritātes."
  • Zem tā izveido numurētu sarakstu (ordered list), kurā uzskaitītas tavas 3 vissvarīgākās iepirkumu lietas.
  • Zem prioritātēm pievieno apraksta sarakstu (description list) ar virsrakstu <h3>, kurā iekļauti vismaz 3 pirkumu apraksti.
  • Izmanto <dt> aprakstāmā priekšmeta nosaukumam un <dd> tā aprakstam.
  • Izmanto papildus <p> tagu zem sarakstiem, lai paskaidrotu, kāpēc pirmais saraksta elements ir vissvarīgākais.

Uzdevums: Kontaktformas izveide

Mērķis: izveidot HTML lapu ar kontaktformu, kurā tiek izmantoti dažādi formas elementi, piemēram, teksta ievades lauki, izvēles rūtiņas, radio pogas un teksta apgabals. Studenti arī apgūs formas piekļuvi un ievades datu organizēšanu.

Norādījumi

  • Izveido pamata HTML lapas struktūru ar <title> "Kontakta forma."
  • Izveido virsrakstu <h1> ar nosaukumu "Sazinies ar mums."
  • Zem virsraksta izveido HTML formu (<form>), kas satur šādus elementus: Teksta lauks vārda ievadīšanai.
  • Teksta lauks e-pasta adreses ievadīšanai.
  • Radio pogas, lai izvēlētos tēmu, piemēram, "Tehniskais atbalsts," "Piedāvājumi" un "Vispārīgi jautājumi."
  • Izvēles rūtiņa, lai piekristu saņemt jaunumus.
  • Teksta apgabals (<textarea>) ziņojuma ievadīšanai.
  • Pievieno pogu "Iesniegt" (<button type="submit">).
  • Pievieno obligāto ievades norādes ar zvaigznīti (*) blakus laukiem, kuriem jābūt obligātiem.

Uzdevums: HTML forma ar dažādiem ievades veidiem

  • Teksta ievades lauks (text)
  • E-pasta ievades lauks (email)
  • Paroles ievades lauks (password)
  • Tālruņa numura ievades lauks (tel)
  • Skaitļu ievades lauks (number)
  • Datuma izvēles lauks (date)
  • Tīmekļa vietnes ievades lauks (url)
  • Atzīmēšanas rūtiņa (checkbox)
  • Radio poga (radio)
  • Faila augšupielādes lauks (file)
  • Iesniegšanas poga (submit)
